How to make a live website using Atom and Github

Definitions

  • Atom - a code editor (created by the same creator of Git)
  • Git Hub - A site that hosts repositories and live sites.
  • Git Hub for Desktop - Allows you to sync your local files to Git Hub and share them as well as view you live site. Sort of like a bridge between what you do on Atom and the live versions online.

Steps!

  1. Download Atom and Git Hub for Desktop. Atom is where you can create and edit your content, Git Hub for Desktop will track changes and allow you to keep your live site up to date.
  2. Make all of your changes in Atom. These will be Highlighted on Git Hub for Desktop as it tracks changes made in yoru local files.
  3. Commit the changes to Git Hub for Desktop, this will lock them in for when you are ready to publish them to Git Hub the website by clicking 'sync'
  4. When you are happy with all of your currently commited changes, click 'sync'.
  5. You can now view your changes live on your Git Hub site.
