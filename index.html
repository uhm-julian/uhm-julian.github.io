How to make a live website using Atom and Github

Definitions

  • Atom - a code editor (created by the same creator of Git)
  • Git Hub - A site that hosts repositories
  • Git Hub for Desktop - Allows you to sync your local files to Git Hub and share them as well as view you live site

Steps!

  1. Make Changes in Atom. These will be Highlighted.
  2. Commit the changes to Git Hub for Desktop.
  3. Sync.
  4. You can now view your changes live on your Git Hub site.
